# Top-level await

### GitHub

{% embed url="https://github.com/tc39/proposal-top-level-await" %}

### Overview

関数宣言や関数式の外（モジュールのトップレベル）でasyncなしのawaitが使えるようになった。

### Example

```javascript
<script type="module">
  const result = await fetch("https://...");
</script>
```
